Parte 1: Princípios de desenvolvimento Web

1.1 História da Web

(Voltar)

Computadores: 1ª geração

Para entender a história da internet, primeiro devemos entender a história dos primeiros computadores. A primeira geração de computadores surgiu com o Harvard Mark I, em 1944. Este consiste em uma grande máquina eletromecânica, com válvulas e criado para gerar cálculos complexos.

Em seguida, foi criado o Colossus, em 1946, com o intuito de criptoanálise na segunda guerra mundial, pela Alemanha. Ao mesmo tempo que nos Estados Unidos, foi criado o ENIAC, já mais automático que o Harvard. Foi nessa época que surgiu o termo "bug", em que ocorria de insetos entrarem nos mecanismos dos computadores, causando erros.

"A simplicidade é a sofisticação máxima." – Leonardo da Vinci

Computadores: 2ª geração

A segunda geração ocorreu entre 1959 e 1965, com a substituição dos computadores eletromecânicos, utilizando válvulas, por computadores utilizando transistores. Nessa época, os computadores deixaram de ocupar uma sala inteira para ocupar uma mesa inteira, sendo mais portáteis.

Computadores: 3ª geração

Essa geração foi marcada pela substituição dos componentes eletrônicos analógicos pelos circuitos integrados envolvendo os transistores, capacitores, resistores em tamanho muito menor. Os tamanhos diminuíram e a eficácia aumentou.

Computadores: 4ª geração

De 1971 até hoje: computadores pessoais. Os circuitos integrados passaram pra microprocessadores, com evolução de design, processadores, memória, etc. Aumento da popularidade e acessibilidade aos computadores. Surgimento de big techs: Apple, Microsoft etc.

Computadores: 5ª geração

Estamos na quinta geração, pela presença de IA, criptomoedas, metaverso e pelo desenvolvimento de computadores quânticos.

A internet evoluiu de forma paralela aos computadores. O primeiro sistema de comunicação foi o telégrafo, com código Morse. Em 1958 foi traçado um cabo intercontinental para a comunicação via telégrafo. Na década de 1950, com a guerra fria, houve muito desenvolvimento de tecnologias, para cálculos complexos como estratégia de guerra: cálculo de rota de mísseis, baseado em muitas variáveis. Nessa mesma época nasceu a DARPA, agência de inteligência avançada, utilizando o ENIAC. Joseph Licklider, teorizou sobre uma rede onde todos poderiam se comunicar, plantando a semente da internet dentro da DARPA. Depois de muito evolução, surgiu a Arpanet, um sistema de comunicação em pacotes, funcionando com “checkpoints”, similares aos roteadores modernos. Em 1969, houve a primeira conexão pela Arpanet, entre a universidade UCLA e SRI. Nesse momento a comunicação era mais voltada para pesquisa científica e planos militares. Com o Vinton Cerf a Robert Kahn houve a criação do TCP/IP, uma atualização na Arpanet. Possibilitando “subnets” em organizações específicas. Em 1898, Tim Berners-Lee, percebendo a crescente de utilização da internet, criou o padrão “WWW”, com base no TCP/IP. Criando o primeiro navegador. Popularizando a internet pela primeira vez. Em 1991, publicou a primeira página web. 1

1.2 O que são Clients?

(Voltar)

Clients, no linguajar da web, é quando a informação é consumida. Exemplos de clientes: notebooks consumindo informações da internet. Na web, o cliente consome arquivos HTLM, CSS, Javascript, Imagens... todos esses são arquivos estáticos. O termo cache, é quando o navegador guarda arquivos já carregados uma vez, para melhorar a velocidade de conexão do cliente.

Exemplo: Um client é um dispositivo como seu computador ou celular que acessa a internet.

1.3 O que são Servers?

(Voltar)

Servers, é onde as informações são armazenadas, recebem requisições e transmitem informações para mais de uma máquina. Há servidores de arquivos, segurança, streaming, e-mail e web. Um conjunto de servidores é chamado de datacenter, onde abastecem grandes redes que possuem muito acessos.

Há partes digitais e físicas de servidores. Entre os tipos de servidores softwares temos sistemas operacionais e servidores web, como o Windows Server, Ubuntu, Apache e Nginx.

O cliente envia requisições HTTP e o servidor retorna HTML, CSS, Javascript, etc. O servidor faz o processamento das informações e retorna de acordo com as requições.

Exemplo: Ao digitar URL no navegador, você está pedindo dados a um servidor.

1.4 O que são Linguagens de programação?

(Voltar)

Linguagem de programação, por definição, é um conjunto de instruções e regras usadas para gerar programas (software). Entre as linguagens mais populares então Javascript, Python e Java.

Há linguagens que rodam no lado do cliente e outras que rodas no lado do servidor.

Entre as que rodam no lado do servidor são: PHP, C#, Java, JavaScript, Ruby.

Entre as que rodam no lado do cliente são: JavaScript.

HTML, no entanto, não é considerada uma linguagem de programação.

HTML é um arquivo de marcação. Um arquivo de texto com delimitadores que o navegador consegue interpretar. Cada delimitador tem uma função e comportamento específico.

Parte 2: Primeiros passos com HTML

2.1 Formulários HTML

(Voltar)

Formulários em HTML são estruturas usadas para coletar dados do usuário e enviá-los para um servidor ou processá-los dentro da própria página. Eles são criados com a tag

e contêm diferentes campos de entrada (inputs), como caixas de texto, botões, menus de seleção, caixas de marcação, entre outros.

Formulários permitem interação entre usuário e aplicação web, sendo fundamentais para cadastros, login, pesquisa, comentários, compras online e muito mais.

2.2 Estrutura do HTML

(Voltar)

A estrutura básica de um documento HTML segue um padrão que organiza o conteúdo da página para que o navegador consiga interpretá-lo corretamente. Ela é composta por tags principais, que delimitam as partes do documento.2

  1. <!DOCTYPE html>: Informa ao navegador que o documento está usando HTML5.
  2. <html lang="pt-BR"...> Tag raiz que envolve todo o documento HTML.
  3. <head>:Contém metadados da página (informações que não aparecem diretamente na tela).
  4. <body>:Contém todo o conteúdo visível da página: textos, imagens, links, tabelas, listas, formulários etc.

2.3 Mídias utilizando HTML

(Voltar)

No HTML é possível utilizar diferentes tipos de mídias para enriquecer as páginas. Podemos inserir imagens em formatos como JPG, PNG, GIF, SVG e WebP, adicionar vídeos em MP4, WebM ou Ogg, além de áudios em MP3, Ogg e WAV. Também é possível incorporar conteúdos externos, como vídeos do YouTube ou mapas do Google Maps, por meio de iframes.

Para gráficos, o HTML oferece suporte a SVG, que s0ão vetores escaláveis, e ao elemento Canvas, que permite criar desenhos, gráficos e animações usando JavaScript. Assim, o HTML possibilita trabalhar com imagens, sons, vídeos, gráficos e conteúdos externos de forma integrada e interativa.

2.4 HTML semântico

(Voltar)

HTML semântico é a prática de escrever código HTML utilizando tags que possuem significado, deixando claro o papel de cada conteúdo na página. Em vez de usar apenas div ou span para tudo, usamos elementos específicos, como:

  • <header> : cabeçalho
  • <nav> : navegação
  • <main> : conteúdo principal
  • <section> : seleção de conteúdo
  • <article> : artigo
  • <aside> : conteúdo complementar
  • <footer> : rodapé

As vantagens dessas tags são: melhora na acessibilidade, favorecimento do SEO e código mais organizado, legível e fácil de manter.
